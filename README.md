# Electronics Parts Web App

A simple web application built with Python and Flask.

## Features

- Search electronic parts
- Filter by category
- View part detail pages
- Display price and stock information

## Tech Stack

- Python
- Flask
- HTML
- CSS
- Git

## How to Run

1. Install Flask

2. Run the application

3. Open in browser
